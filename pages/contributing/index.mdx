# 贡献指南

欢迎来到 Westlake for Good 贡献指南！我们非常感谢您的参与和贡献。本指南旨在帮助您了解如何为我们的平台做出贡献。

## 贡献方式

您可以通过以下几种方式参与贡献：

1. **内容贡献**：提交新的AI提示词示例或AI安全研究案例
2. **文档改进**：优化现有文档，修正错误，或提升文档质量
3. **功能开发**：为平台开发新功能或改进现有功能
4. **问题反馈**：报告问题或提出改进建议

## 贡献流程

我们采用GitHub的Fork-PR工作流程：

1. **Fork仓库**：将我们的GitHub仓库Fork到您的账户
2. **创建分支**：在您的Fork仓库中创建一个新分支
3. **修改内容**：在新分支上进行您的修改
4. **提交PR**：向主仓库提交Pull Request
5. **审核流程**：等待社区成员审核您的贡献
6. **合并修改**：通过审核后，您的贡献将被合并到主分支
