# 科学文献总结助手

## 简介

这个提示词设计用于帮助研究人员快速提取和总结科学论文的关键信息。它能够识别论文的主要发现、方法、结论和局限性，并以结构化的方式呈现。

## 提示词

```
请你作为一名专业的科学文献分析专家，帮我分析以下科学论文摘要或章节内容。请按照以下结构输出分析结果：

1. 研究目标：用1-2句话概括研究旨在解决什么问题
2. 主要方法：简要描述研究采用的主要方法或技术路线
3. 关键发现：列出3-5个最重要的研究发现或结果
4. 创新点：指出该研究的创新之处
5. 局限性：分析研究可能存在的局限性
6. 未来方向：根据论文内容推断可能的未来研究方向
7. 相关领域：指出2-3个与该研究紧密相关的研究领域

请保持客观、准确，避免添加原文中未包含的信息。如果原文中某部分信息缺失，请在相应部分注明"原文未明确提及"。

论文内容如下：
[在此粘贴论文摘要或章节内容]
```

## 示例输出

```
基于提供的论文摘要分析如下：

1. 研究目标：研究旨在开发一种新型深度学习架构(DualGCN)，用于提高自然语言处理任务中的方面级情感分析准确性。

2. 主要方法：该研究提出了一种基于图卷积网络的双重注意力机制，同时建模依存树和完全图结构，并设计了特定的交互方式来融合句法信息和全局上下文信息。

3. 关键发现：
   - DualGCN在六个公共基准数据集上实现了最先进的性能
   - 句法信息和全局上下文的结合对模型性能有显著提升
   - 模型在复杂句子结构的情感分析方面表现尤为出色
   - 注意力机制有效地捕获了句子中与情感相关的关键元素

4. 创新点：提出了一种新颖的双重图卷积网络架构，能够同时处理依存句法结构和全局词汇关系，并设计了两种图结构之间的特定交互机制。

5. 局限性：原文未明确提及局限性，但可能包括计算复杂度较高、对高质量句法分析的依赖性等。

6. 未来方向：研究可能的扩展方向包括模型在多语言环境下的应用、处理更复杂的情感表达（如讽刺、隐喻）以及探索更高效的图结构交互方式。

7. 相关领域：
   - 基于图神经网络的自然语言处理
   - 情感分析与观点挖掘
   - 句法感知的深度学习模型
```

## 使用说明

1. 将您需要分析的科学论文摘要或章节复制到提示词中的"[在此粘贴论文摘要或章节内容]"部分
2. 根据需要，您可以适当调整分析结构，例如添加"方法创新"、"数据集"等额外部分
3. 对于较长的论文，建议分段提交，每次处理一个主要部分（如摘要、方法、结果等）
4. 如果论文涉及特定领域术语，可以在提示词开头指明领域，如"作为一名人工智能/生物学/物理学专家..."
